# customer-shopping-behavior-analysis


End-to-end data analysis project to understand customer purchasing behavior and identify revenue growth and retention opportunities.

**Tools:** Python (Pandas), SQL, Power BI  
**Dataset:** ~3,900 customer shopping records  

**Key Insights:**
- Clothing is the top revenue-generating category  
- Subscription customers spend more on average  
- Customer behavior varies by age, gender, and season  

**Output:** Interactive Power BI dashboard + business-focused analysis report
